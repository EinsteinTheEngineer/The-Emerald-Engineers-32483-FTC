# FTC Into The Deep - Fully Autonomous Complete Package

This single zip archive includes absolutely everything required to run your visual path auto-coding interface seamlessly without requiring any external files or assets.

## Files Extracted inside this Folder:
1. `index.html` — The core visual mapping app.
2. `field_setup.jpg` — The exact "Into The Deep" arena map layout image you uploaded.
3. `README.md` — This instruction set.

## Quick Start Instructions:
1. Extract the entirety of this `.zip` package into its own folder on your desktop.
2. Double-click `index.html`. 
3. The interface will immediately display your complete "Into The Deep" field layout directly onto the visual drawing canvas with translucent tile overlays.
4. Left-click anywhere on the canvas grid mapping to lay coordinates. The box on the right will auto-write a complete Java encoder implementation class.
5. Click **"Copy Java Code"** to put it straight into your Android Studio `teamcode` package directory!
